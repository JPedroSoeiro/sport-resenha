🎮 O Jogo
O simulador permite que 3 jogadores escolham times reais do Brasileirão e tentem reconstruir seus elencos através de rodadas de contratações, vendas e leilões.

Regras Principais:
Orçamento Inicial: Definido pelo Host no início da partida para todos os times.

Venda de Jogadores: Ao vender um titular, o valor de mercado é adicionado ao caixa. O jogador vendido sai do jogo (não vai para o mercado aberto).

Rodadas de Posição: O jogo avança por posições (Goleiro, Lateral, Zaga, etc.). Em cada rodada, 3 opções de contratação são oferecidas (incluindo jogadores europeus viáveis entre 15M€ e 25M€).

O Leilão e a Punição do Reserva: * Se dois jogadores quiserem a mesma contratação, inicia-se um leilão manual.

Quem perder o leilão é proibido de contratar naquela rodada.

O time perdedor deve, obrigatoriamente, utilizar o seu Jogador Reserva para aquela posição.
